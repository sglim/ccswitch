Convenience slash commands available in this folder (Claude-Code-only):
- `/usage` — utilization table
- `/switch` — switch to picker's recommendation
- `/list` — list managed accounts
- `/add` — add the currently-active account
- `/handicap` — set per-account handicap
- `/agent` — LaunchAgent state
