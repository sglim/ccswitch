description: ccswitch.sh 전체 명령어 reference.

`ccswitch.sh --help` 실행, 출력 그대로 보여줄 것.

사용자가 특정 명령에 대해 물으면 README 의 "명령어" 표와 "Picker 알고리즘" 섹션 참조.

이 폴더에서 쓸 수 있는 Claude Code 전용 slash 명령:
- `/usage` — 사용량 표
- `/switch` — picker 추천 계정으로 전환
- `/list` — 관리 중인 계정 목록
- `/add` — 현재 active 계정 등록
- `/handicap` — 계정별 handicap 설정
- `/agent` — LaunchAgent 상태
